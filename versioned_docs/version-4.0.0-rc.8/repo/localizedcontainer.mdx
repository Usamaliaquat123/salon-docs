A Salon Element for displaying a container with RTL (Right to left) feature.

LocalizedContainer displays the content with in a row of content

```tsx
import { TextProp } from "react-native";
import { LocalizedContainer } from "@mobileElements/index";

<LocalizedContainer style={ /** Style Object**/}>
        /** Content here  **/ 
</LocalizedContainer>;
```
