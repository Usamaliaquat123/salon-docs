## Installation

## Quick start

```ts
yarn add @umar-epacken/salon-data
```

This is served as a private npm package.

## Installation

1. **.npmrc File:** You will need an `.npmrc` file to have it installed in your project.

2. **Environment Variables:** Double check the env variables in `env.ts` are all set as requested by the package.

3. **Install Dependencies:** Execute `yarn` to install the dependencies.

4. **.env file** make sure your env file should be proper defined.


## ```tsx env ``` Sample

```tsx 
IS_WEB= //BOOLEAN | string
BASE_URL= //API_ENDPOINT | string
API_ENVIRONMENT= //enviroment  | JSON | REST -> string
NEXT_PUBLIC_IS_WEB = // if is using NEXT enviroment
VITE_IS_WEB = // if is using VITE enviroment 

```

Includes try-catch blocks within each API function in a hook to handle errors effectively.
